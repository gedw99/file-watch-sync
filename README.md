# file-watch-sync


cli tool to configure directory to watch using a PUB SUB pattern.


## Building

``` 
make install
``` 

## Usage:

``` 
go install github.com/gedw99/file-watch-sync@latest
``` 

## Examples

In example folder:

``` 
make run
```
